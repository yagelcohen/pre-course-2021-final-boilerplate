# ToDo-List

ToDo list is a simple web application to save Your daily tasks in order.

### Features
* Pick a date.
* Add new task.
* Mark task as completed / uncompleted By line on the task.
* Set task priority抓
* Edit task抓
* Delete single task抓
* A search button that can search for the existing task抓
* A button that sorts the tasks in order of importance (five is the highest to one which is the smallest)抓
* The information is saved in the localStorage To use it later even if you refresh the page.
* Does not allow you to create a new task if you have not filled all the information, Then sends an alert to notify the user抓
